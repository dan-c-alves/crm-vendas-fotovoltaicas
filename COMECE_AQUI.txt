📋 REQUISITOS ANTES DE COMEÇAR
═══════════════════════════════════════════════════════════════════════════

1. Python 3.9+ instalado
   - Download: https://www.python.org/downloads/
   - Marque "Add Python to PATH" durante a instalação

2. Node.js 18+ instalado
   - Download: https://nodejs.org/
   - Instale a versão LTS (Long Term Support)

3. VS Code (opcional, mas recomendado)
   - Download: https://code.visualstudio.com/


🚀 INICIAR O CRM
═══════════════════════════════════════════════════════════════════════════

OPÇÃO 1: Iniciar Automático (RECOMENDADO)
──────────────────────────────────────────

Windows 11 Pro:
  1. Duplo clique em: INICIAR.bat
  2. Aguarde a inicialização (pode levar 1-2 minutos)
  3. Abra o navegador em: http://localhost:3000

macOS/Linux:
  1. Abra o terminal
  2. Execute: ./INICIAR.sh
  3. Abra o navegador em: http://localhost:3000


OPÇÃO 2: Iniciar Manual
──────────────────────────

Backend (Terminal 1):
  1. cd backend
  2. python -m venv venv
  3. venv\Scripts\activate  (Windows) ou source venv/bin/activate (Mac/Linux)
  4. pip install -r requirements.txt
  5. python main.py
  
  ✓ Backend em: http://localhost:8000

Frontend (Terminal 2):
  1. cd frontend
  2. npm install
  3. npm run dev
  
  ✓ Frontend em: http://localhost:3000


🌐 ACESSAR O CRM
═══════════════════════════════════════════════════════════════════════════

Local (seu computador):
  - Dashboard: http://localhost:3000
  - API: http://localhost:8000
  - Docs API: http://localhost:8000/docs

Telemóvel (mesma rede):
  - Dashboard: http://192.168.1.185:3000
  (Substitua 192.168.1.185 pelo IP do seu computador)

💾 MIGRAR DADOS ANTIGOS
═══════════════════════════════════════════════════════════════════════════

Se tem dados antigos (104 leads, 16 vendas):

  1. Copie o ficheiro crm_database.db para: data/
  2. No terminal, execute:
     cd backend
     python scripts/migrate_data.py
  
  ✓ Dados migrados automaticamente!

❓ DÚVIDAS FREQUENTES
═══════════════════════════════════════════════════════════════════════════

P: Erro "Port already in use"
R: A porta já está em uso. Feche outros programas ou altere a porta.

P: Erro "ModuleNotFoundError"
R: Certifique-se que o ambiente virtual está ativado.

P: Erro "CORS error"
R: Verifique se o backend está em execução em http://localhost:8000

P: Como aceder no telemóvel?
R: Use o IP do seu computador. Ex: http://192.168.1.185:3000

P: Como mudar a percentagem de comissão?
R: Edite: backend/config/settings.py (COMISSAO_PERCENTAGEM)
